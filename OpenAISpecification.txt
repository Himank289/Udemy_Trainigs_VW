	collection/item/colection is valid
	courses/{courseId}/students is valid

Step 21: Identify the API Versioning Scheme and Set the API Version

Versioning API Approaches :

1. No Versioning  -- changes will occur directly in the existing api
2. URL Versioning -- /api/v1/colleges/10
3. Query String Versioning --/api/colleges/10?version=1
4. Header Versioning -- Custom header added to indicate version X-API-Version=2
5. Media Type Versioning -- Content-Type:application/vnd.cms.v1+json
